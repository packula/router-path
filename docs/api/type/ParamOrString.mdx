# `ParamOrString`

A utility type for use when you need to be able to accept any valid [`Param`]
type or a string.

```ts no-transpile
import {ParamOrString} from '@snout/router-path'
```

Functionally equivalent to:

```ts no-transpile
Param<string, any> | string
```
