# `AnyParam`

A utility type for use when you need to be able to accept any valid [`Param`]
type.

[`param`]: Param.mdx

```ts no-transpile
import { AnyParam } from "@snout/router-path";

function logParam(param: AnyParam): void {
  console.log(param.name);
}
```

Functionally equivalent to:

```ts no-transpile
Param<string, any>;
```
